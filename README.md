# Water Cycle Vocabulary Adventure

A kid-friendly React, TypeScript, Tailwind CSS, and Vite vocabulary game for elementary water cycle words.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL Vite prints, usually:

```text
http://localhost:5173/
```

## Build

```bash
npm run build
```

## Teacher Area

Use passcode:

```text
teacher123
```

Progress, stars, badges, and completed words are saved in `localStorage`. The Teacher Area includes a reset button for student progress.
